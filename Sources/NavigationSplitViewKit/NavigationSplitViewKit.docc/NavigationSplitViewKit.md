# ``NavigationSplitViewKit``

Build responsive three-column layouts with synchronized state for iOS, iPadOS, and macOS.

## Overview

NavigationSplitViewKit demonstrates how to build a robust multi-column interface using SwiftUI's `NavigationSplitView`. It handles column visibility, selection synchronization, and size class adaptations across all Apple platforms.
